# Acharya Ventures 2 Landing Page

All landing pages in one folder for easy deployment.

## Structure

| Folder | Description |
|--------|-------------|
| **Avlanding page 1** | Acharya Ventures - Build with Clarity (terracotta/peach theme) |
| **Avlanding page 2** | Acharya Ventures - Human-Centric Startup Growth (green theme) |
| **gurukullanding page 1** | Startup Gurukul - Main landing |
| **gurukullanding page 2** | Startup Gurukul - Design 2 |
| **gurukullanding page 3** | Startup Gurukul - Design 3 |

## Run Locally

```bash
npm start
```

Then open:
- **Avlanding page 1:** http://localhost:8080/Avlanding%20page%201/
- **Avlanding page 2:** http://localhost:8080/Avlanding%20page%202/
- **gurukullanding page 1:** http://localhost:8080/gurukullanding%20page%201/
- **gurukullanding page 2:** http://localhost:8080/gurukullanding%20page%202/
- **gurukullanding page 3:** http://localhost:8080/gurukullanding%20page%203/

## Deploy

Upload the entire folder to any static host (Netlify, Vercel, GitHub Pages, etc.). Each subfolder is a standalone deployable landing page.
